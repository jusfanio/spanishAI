An interactive web application for learning Spanish vocabulary, verb conjugations, and testing your knowledge.

## Features

- **Flashcards**: Study 30+ Spanish words with translations and examples
- **Quiz Mode**: Test your knowledge with multiple-choice questions
- **Verb Conjugation**: Practice conjugating common Spanish verbs in multiple tenses
- **Progress Tracking**: Monitor your learning progress with saved statistics

## How to Deploy and Share

### Option 1: Netlify (Easiest - Recommended)

1. Go to [netlify.com](https://www.netlify.com) and sign up (free)
2. Drag and drop the entire `assets-hub` folder onto Netlify's deploy area
3. Your site will be live instantly with a URL like `your-site-name.netlify.app`
4. Share the URL with friends!

### Option 2: Vercel

1. Go to [vercel.com](https://vercel.com) and sign up (free)
2. Click "New Project"
3. Drag and drop the `assets-hub` folder
4. Your site will be live with a URL like `your-site-name.vercel.app`

### Option 3: GitHub Pages

1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings â†’ Pages
4. Select "main" branch and "/ (root)" folder
5. Your site will be at `your-username.github.io/repository-name`

### Option 4: Surge.sh (Command Line)

```bash
npm install -g surge

# Follow the prompts to create your site
```

## Local Development

Simply open `index.html` in your web browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`
